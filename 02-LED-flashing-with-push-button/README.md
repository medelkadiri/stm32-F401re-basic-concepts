# **Project Description**



* This project enables flashing the LED using a push button integrated in the STM32 Nucleo-64 development board with the STM32F401RE MCU.



# **Main Points of the program**



* GPIOA Pin 5 configured for LED output.



* GPIOC Pin 13 configured for push button Input(GPIOC Pin 13).



* Flash the LED only if the push button is triggered(if the condition on the value read in the GPIOC pin 13 is met).



* Verified functionality on the target board.
