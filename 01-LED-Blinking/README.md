# **Project Description**



* This project enables control over LED's blinking rate integrated in the STM32 Nucleo-64 development board with the STM32F401RE MCU.



# **Main Points of the program**



* GPIOA Pin 5(PA5) configured for LED output.



* Toggle LED with HAL Library Toggle function.



* Control  LED's blinking rate with a delay function implemented with a for loop.



* Verified functionality on the target board.
